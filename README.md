# The-Last-Drop

The last drop is a 2D tetris-like game developed as a second semester solo project.

# Gameplay preview (Click on image)

[![Gameplay Video](Backgrounds/mq3.webp)](https://youtu.be/ZJvTlS2WGrk)

 # Tech Stack
 
- C++
- Object Oriented Programming
- SFML

# Gameplay

- To complete a line fill every space in that line.
- Complete 10 lines to finish a level.
- Total 8 levels to complete.
- Game speed increases as levels go on.
- Keyboard-based controls

# Project Context

Built individually as a game development project for the Object Oriented Programming course.

To run game:
Copy all files out of subfolders into a main folder and place the new folder in the compiler/IDE directory
